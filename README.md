# Javascript_Challenge
This application has two parts.

Download all the files to the local machine and run the application.

UFO Level 1
First part UFO-Level1 displays the complete ufo sighting dataset in a tabular form on the web page. 
It allows to filter based on the date entered. The data matching the specified date will be displayed.

UFO Level 2
The webpage will display the entire dataset as a table by default. It allows to set multiple filters and search for UFO sightings using date/time, city, state, country and shape criteria.
